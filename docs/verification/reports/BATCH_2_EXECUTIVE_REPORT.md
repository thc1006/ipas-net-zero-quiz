**問題描述**:
```
原題:
```

**IPCC AR6 實際內容**:
- Working Group II 討論溫度變化的非線性影響
- 不同環境衝擊具有不同的物理/生物響應曲線

**修正建議** (二選一):

**方案 A** (推薦 - 提高準確性):
```
AR6 指出：地表溫度每上升 0.5°C，對融冰、海平面、物種減少等的影響將呈現：
A: 線性增加
B: 加速增加  ← 改選此項
C: 指數級加速
D: 完全消失

理由: 更準確反映 AR6 的"非線性放大"結論
來源: AR6 WG II Chapter 3 & 4
```

**方案 B** (保留並補充出處):
```
保留原題與答案，但補充解析:
「根據 IPCC AR6 評估，溫度變化對不同環境系統的影響呈加速度惡化。
 源自 AR6 Working Group II Report, Section X.X (具體章節待補)」
```
