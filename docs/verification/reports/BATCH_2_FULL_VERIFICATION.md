## 疑慮項詳細分析

### ⚠️ c2-077: UNEP 2023 排放差距報告數值

**原題**:
```
根據 UNEP《2023 排放差距報告》，現行 NDC 估計本世紀末將升溫多少？
答案: C (3–3.2°C)
```

**驗證發現**:
- UNEP EGR 2023 原文表述:
  - 若 NDC 完全兌現: 2.5-2.9°C
  - 若維持現行政策: 3°C
  - 若現有承諾生效: 2.6-2.7°C

**疑慮來源**:
- 題目提供的"3-3.2°C"不是報告中的精確數值
- 報告未明確使用此區間

**建議**:
- ✅ 如果題目意圖是"接近 3°C"，則可接受
- ⚠️ 若要求精確數值，建議改為 "2.5-2.9°C" 或 "維持現行政策下約 3°C"
- 嚴格評分: 細微誤差，實質內容正確

### ⚠️ c2-089: AR6 溫度影響次方成長

**原題**:
```
AR6 指出：地表溫度每上升 0.5°C，對融冰、海平面、物種減少的影響呈何次方成長？
答案: C (10 次方)
```

**驗證發現**:
- AR6 WG II (2022) 討論了溫度變化的非線性影響
- 但報告中不存在明確的"10 次方"數學表述
- 不同影響（融冰、海平面上升、物種滅絕）有不同的響應曲線
- 這是對複雜科學的過度簡化

**疑慮來源**:
- 缺乏 AR6 原始出處的明確引用
- "次方成長"表述過度抽象
- 與 IPCC 審慎用語習慣不符

**建議**:
- ⚠️ 此題表述不精確
- 建議改為: "AR6 指出溫度每上升 0.5°C，不同環境衝擊的惡化程度將"
- 或提供 AR6 具體出處頁數
