**題目**: 產品碳足跡盤查流程與碳標籤申請，下列敘述何者錯誤？

#### c2-184: 科學方法優先順序 ✅

**題目**: 產品碳足跡科學方法優先順序，下列敘述何者錯誤？

#### c2-185: 產品生命週期階段 ✅

**題目**: 以下何者不屬於產品生命週期階段？
**答案**: B (研發)
**驗證結果**: ✅ **正確**

**結論**: B2B 產品包含原料取得階段，選項 B 正確。
