### 🔴 【緊急】c1-001: ISO 14064-1:2018 類別4分類

**錯誤類型**: 系統性標準誤解

**當前內容**:
```json
{
  "id": "c1-001",
  "question": "依據 ISO 14064‑1:2018 標準，下列何者「屬於」其他間接排放（類別 4）的範疇？",
  "options": {
    "A": "商務差旅的排放",
    "B": "租賃資產的排放",
    "C": "員工通勤的排放",
    "D": "購入的電力使用"
  },
  "answer": "B",
  "explanation": "根據ISO 14064-1:2018，類別4指的是「由組織使用的產品所產生之間接溫室氣體排放」..."
}
```

**問題分析**:

| 項目 | 現況 | 正確情況 |
|------|------|---------|
| 選項B定義 | 租賃資產 | ❌ 不屬於Category 4 |
| Category 4 | 錯誤列舉 | ✅ 應為外購商品、資本商品、廢物處理 |
| 標準來源 | ISO 14064-1:2018 | ⚠️ 但定義有誤 |

**修正方案A** (推薦 - 最小改動):

```json
{
  "id": "c1-001",
  "question": "依據 ISO 14064‑1:2018 標準，下列何者「屬於」其他間接排放（類別 4）的範疇？",
  "options": {
    "A": "商務差旅的排放",
    "B": "外購商品的排放",
    "C": "員工通勤的排放",
    "D": "購入的電力使用"
  },
  "answer": "B",
  "explanation": "根據ISO 14064-1:2018，類別4指的是「由組織使用的產品所產生之間接溫室氣體排放」，其中包括外購商品、資本商品、上遊租賃資產、外購服務和廢物處理。因此，外購商品的排放屬於類別4。 [1, 2, 3, 4, 5]"
}
```

**修正方案B** (改變答案):

```json
{
  "id": "c1-001",
  "question": "依據 ISO 14064‑1:2018 標準，下列何者「屬於」其他間接排放（類別 3）的範疇？",
  "options": {
    "A": "商務差旅的排放",
    "B": "租賃資產的排放",
    "C": "員工通勤的排放",
    "D": "購入的電力使用"
  },
  "answer": "A",
  "explanation": "根據ISO 14064-1:2018，類別3指的是「組織運輸相關之間接溫室氣體排放」，包括商務差旅、員工通勤等。因此，商務差旅的排放屬於類別3。 [1, 2, 3, 4, 5]"
}
```

**修正方案C** (區分標準框架):

```json
{
  "id": "c1-001",

```

**建議採用**: **方案A** (修改選項為「外購商品」)
- ✅ 保持題目難度

**改進類型**: 解釋精確度

**當前內容**:
```json
{
  "id": "c1-002",
  "question": "關於 GRI 報告框架與 SASB 標準的差異比較，下列敘述何者錯誤？",
  "options": {
    "A": "GRI 涵蓋範圍較廣泛，SASB 則較為聚焦於財務重大性",
    "B": "GRI 僅適用於上市公司，SASB 則適用於所有類型企業",
    "C": "GRI 採用利害關係人導向，而 SASB 主要關注投資人需求",
    "D": "GRI 採用通用性準則，SASB 採用產業別特定準則"
  },
  "answer": "B",
  "explanation": "GRI 報告框架適用於所有類型的組織，而 SASB 則特別針對可能影響企業財務狀況與營運績效的 ESG 議題。 [1, 2, 3, 4, 5]"
}
```

**改進建議** (補充詳細說明):

```json
{
  "id": "c1-002",
  "question": "關於 GRI 報告框架與 SASB 標準的差異比較，下列敘述何者錯誤？",
  "options": {
    "A": "GRI 涵蓋範圍較廣泛，SASB 則較為聚焦於財務重大性",
    "B": "GRI 僅適用於上市公司，SASB 則適用於所有類型企業",
    "C": "GRI 採用利害關係人導向，而 SASB 主要關注投資人需求",
    "D": "GRI 採用通用性準則，SASB 採用產業別特定準則"
  },
  "answer": "B",
  "explanation": "選項B錯誤。實際情況為：GRI報告框架適用於【所有類型組織】，包括上市公司、私營企業、非營利組織、政府機構等；而SASB標準則【主要針對上市公司】及投資相關的財務重大性議題。因此，SASB適用範圍更窄，而非GRI。GRI的通用性和廣泛適用性是其與SASB的主要區別。 [1, 2, 3, 4, 5]"
}
```

**改進原因**:
- 原解釋表述模糊
- 新解釋明確區分兩者適用對象
- 增加學習效果

**驗證資源**:
- https://www.globalreporting.org/standards/
- https://help.sasb.org/
